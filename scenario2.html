Building Evacuation - Blocked Exit

Learning Objective

Learn the correct procedures for evacuating an office building during a fire, especially when a primary exit is blocked.

There is a fire reported on the second floor. Please evacuate the building using the nearest available exit." You smell faint smoke coming from the hallway towards the main stairwell.
